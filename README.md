Pandas Data Analysis Project 📊

📌 Description

This project is a simple data analysis program using Python.
It analyzes student marks and calculates basic statistics like average, highest, and lowest marks.

---

⚙️ Features

- Creates dataset using Python
- Uses Pandas library for analysis
- Calculates average marks
- Finds highest and lowest marks
- Displays data in table format

---

🛠️ Technologies Used

- Python
- Pandas library

---

▶️ How to Run

1. Install pandas:
   pip install pandas

2. Run the program:
   python analysis.py

---

📷 Sample Output

Data:
Name   Marks
A      80
B      75
C      90
D      60

Average Marks: 76.25
Highest Marks: 90
Lowest Marks: 60
